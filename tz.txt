
Тестовое задание
Реализовать простое приложение таск трекера с управлением задачами через API.

Приложение должно содержать:

    Задачи
    Список пользователей на которых можно назначить задачу


Возможные действия через API:

    Создание задачи
    Удаление задачи
    Редактирование задачи
    Получение списка задач с различными наборами фильтров
    Изменить статус задачи
    Изменить пользователя на которого назначена задача

Технические требования

    Приложение должно быть написано на PHP.
    Приложение не должно быть написано с помощью какого-либо фреймворка, однако можно устанавливать для него различные пакеты через compоser.
    Результат API запросов должен быть в виде JSON.
    Результат задания должен быть выложен на github, должна быть инструкция по запуску проекта. Также необходимо пояснить, сколько на каждую часть проекта ушло времени.
    Весь код должен быть прокомментирован в стиле PHPDocumentor'a.
    Используется база данных Mysql.
